Лабораторная работа 4. Алгоритмы работы со строками

Цель работы:научиться разрабатывать скрипты на языке JavaScript, реализующие алгоритмы обработки строковых данных с использованием свойств и методов объекта String.

Задача №1

Формулировка условия:

Даны две строки s1 и s2. Посчитать, сколько раз в строке s2, встречается первый символ строки s1.

Результат:

Параметр Значение
Количество вхождений -

Тесты

Тест 1:

Входные данные: s1 = "apple", s2 = "banana apple apple"

Ожидаемый результат: 2

Тест 2:

Входные данные: s1 = "a", s2 = "apple"

Ожидаемый результат: 0

Тест 3:

Входные данные: s1 = "x", s2 = "hello world"

Ожидаемый результат: 1
